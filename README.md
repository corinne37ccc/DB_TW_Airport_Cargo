# DB_TW_Airport_Cargo

台灣機場貨運量分析儀表板
Taiwan Airport Air Cargo Statistics Dashboard
資料範圍：2015–2025年｜資料來源：交通部民航局統計月報

Tab 1｜全國總覽
全台各機場歷年貨運量（總量、進口、出口、轉口）趨勢與各機場貨量佔比。
Tab 2｜桃園機場 — 貨量總體分析

子頁籤「總覽」：桃園機場歷年貨量趨勢與進出口結構分析。
子頁籤「貨運站比較」：華儲、遠翔、長榮等各貨運站歷年處理量比較與交叉分析。

Tab 3｜桃園機場 — 進出口細項分析

子頁籤「進口貨品分析」：依貨品類別與來源國分析進口貨量趨勢與年度佔比。
子頁籤「出口貨品分析」：依貨品類別與目的國分析出口貨量趨勢與年度佔比。
子頁籤「交叉分析」：貨品類別 × 國家雙維度交叉分析（進口或出口可切換）。

Tab 4｜桃園機場 — 貨運月報
桃園機場月度貨量趨勢、各貨運站月度比較，及國籍航空公司月度貨量。
Tab 5｜其他國際機場貨運量分析
高雄、松山、台中機場歷年貨運量趨勢與進出口結構比較。
Tab 6｜國籍航空公司貨量分析
國籍航空公司（各航空公司可多選）歷年貨量趨勢、各機場分布及貨量組成。
